## 1. 什么是GraphQL

- GraphQL既是一种用于API的查询语言,也是一个满足你数据查询的运行时
- GraphQL对你的API中的数据提供了一套易于理解的完整描述,使得客户端能够准确的获得他需要的数据,而且没有任何冗余
- 请求你所要的数据不多不少
- 只用一个请求获取多个资源
